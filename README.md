# Sahaay – Mental Health Support Platform

Sahaay is a mental health support platform designed to provide users with a safe, accessible, and supportive environment to improve their emotional well-being. It combines technology with empathy to offer guidance, resources, and community support.

---

## 🚀 Features

- **AI Chat Support**
  Interactive chatbot to help users express feelings and receive guidance.

- **Mood Tracking**
  Track daily emotions and visualize mental health patterns.

- **Resource Library**
  Articles, exercises, and coping strategies for mental wellness.

-  **Community Support**
  Safe space for users to share and connect with others.

- **Reminders & Notifications**
  Daily check-ins and wellness reminders.

- **Secure & Private**
  User data is protected with authentication and privacy measures.

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/sahaay.git
cd sahaay
```

### 2. Install dependencies

```bash
# Frontend
cd frontend
npm install

# Backend
cd ../backend
npm install
```

### 3. Setup environment variables

Create a `.env` file in the backend folder:

```env
PORT=5000
MONGO_URI=your_database_url
API_KEY=your_ai_api_key
JWT_SECRET=your_secret_key
```

### 4. Run the project

```bash
# Backend
npm run dev

# Frontend
npm start
```
